Game rules

How scoring works

This is a Fantasy Football-style draft for Big Brother. Each entry ranks every houseguest from most likely to win to least likely to win. Scores are updated from competition outcomes, eviction order, and final-season bonuses.

The code remains published for transparency, and the static site can be regenerated after each update with the Python generator.

Point scoring
  1. Head of Household: 10 minus the contestant ranking.
  2. Veto: 10 minus the contestant ranking.
  3. Getting off the block: 7.5 minus 0.75 times ranking.
  4. Other competitions: 5 minus 0.5 times ranking.
  5. Buy back: 10 minus ranking when an evicted houseguest returns.
  6. Perfect eviction order: 100 bonus points.
  7. America's Favorite, winner, runner-up, and third-place bonuses or penalties depend on where they appeared in each list.
Recent rule changes
  • A perfect eviction-order prediction now earns a bonus.
  • Only one ranking slot now earns zero points.
  • Getting off the block is worth slightly less than prior years.
  • Returning evicted houseguests can add or subtract points.
  • America's Favorite is worth more than before.
