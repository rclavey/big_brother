Game rules

How scoring works

This is a Fantasy Football-style draft for Big Brother. Each entry ranks every houseguest from most likely to win to least likely to win. Scores are updated from competition outcomes, eviction order, and final-season bonuses.

The code remains published for transparency, and the standings will be updated after each episode (or as soon as I can; I'm only one person, but I promise at least once per week :).

Point scoring
  1. Head of Household: 10 minus the contestant ranking.
  2. Veto: 10 minus the contestant ranking.
  3. Getting off the block: 7.5 minus 0.75 times ranking.
  4. Other competitions: 5 minus 0.5 times ranking.
  5. Buy back: 10 minus ranking when an evicted houseguest returns.
  6. Perfect eviction order: 100 bonus points.
  7. America's Favorite: +75 if ranked 1st, +50 if ranked 2nd, +25 if ranked 3rd, -10 if ranked third-to-last, -25 if ranked second-to-last, and -50 if ranked last.
  8. Season winner: +100 if ranked 1st, +75 if ranked 2nd, +50 if ranked 3rd, -25 if ranked third-to-last, -50 if ranked second-to-last, and -75 if ranked last.
  9. Runner-up: +75 if ranked 1st, +50 if ranked 2nd, +25 if ranked 3rd, -10 if ranked third-to-last, -25 if ranked second-to-last, and -50 if ranked last.
  10. Third place: +50 if ranked 1st, +25 if ranked 2nd, +10 if ranked 3rd, -5 if ranked third-to-last, -10 if ranked second-to-last, and -25 if ranked last.
Recent rule changes
  • A perfect eviction-order prediction now earns a bonus.
  • Only one ranking slot now earns zero points.
  • Getting off the block is worth slightly less than prior years.
  • Returning evicted houseguests can add or subtract points.
  • America's Favorite is worth more than before.
